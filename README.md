# SIRE – Simulated Incident Response Environment

> Real-time, multi-user emergency training simulator built with Node.js, Express, Socket.IO, and React.

📄 **[View the landing page →](https://aelayette.github.io/sire-simulator/)**

SIRE delivers live, progressively-escalating incident scenarios to trainees over WebSockets, enabling instructors to run realistic emergency response drills across security, medical, fire, and environmental categories — no database required.

Developed as a capstone project for NSCC's IT Programming program.
